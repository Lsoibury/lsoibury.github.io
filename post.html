如何用 GitHub Pages 搭建个人博客

从零开始，一步步教你用静态文件搭建漂亮的个人博客站点，无需后端服务器。

前言

GitHub Pages 是 GitHub 提供的免费静态站点托管服务。你只需要将 HTML、CSS、JavaScript 文件推送到仓库，即可得到一个公开访问的网站。

准备工作

在开始之前，你需要：

  • 一个 GitHub 账号
  • 基本的 HTML/CSS 知识
  • Git 命令行工具

创建仓库

将仓库命名为 yourusername.github.io，然后将本项目的所有文件推送进去即可。

提示：确保仓库为 Public，否则 GitHub Pages 无法正常工作（免费账户）。

自定义内容

编辑 assets/js/data.js 文件，修改文章数据、个人信息等内容，无需任何构建工具。

// 示例：添加一篇文章
{
  id: 1,
  title: "我的第一篇博客",
  summary: "这是摘要…",
  tag: "生活",
  date: "2024-06-01"
}

就这么简单！修改保存后推送到 GitHub，几分钟内即可看到效果。
